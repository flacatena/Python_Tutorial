Django Model

Data is typically stored in a database as tables with rows and columns. Django implements tables with the use of Python classes. The Model component of the MVT architecture is implemented in Django by using these Python classes. A Model in Django is essentially a representation of a table in a database with the attributes of the class representing a table’s fields. Django’s Object Relational Mapper (ORM) is used as an intermediary to perform database operations which makes it possible for a developer to use Python objects instead of SQL queries.
